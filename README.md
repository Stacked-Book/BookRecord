## ✅ 방식
- 매주 금요일 오후10시 디스코드 온라인 회의
- 1주단위로 책을 정리한다.
- 분량은 주당 책의 1장~2장 정도로 유동적으로 정한다.
- 스터디 시간 전까지 각자 정리한자료를 업로드합니다.
- 스터디당일 모여서 발표자 랜덤선정후 발표하고 토론,질의응답,자료공유 하는 시간을 가진다. (모임o)
